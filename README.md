# Agent 边界图鉴

一个面向中文用户的 AI Agent 对比网站。V1 收录 10 款产品，以运行环境、工具权限、审批机制、自主程度、持续运行和交付方式为统一维度，帮助用户先看边界，再判断产品是否适合自己的目标。

- 主站：https://agent-boundary-atlas-cn.tinalexu0101.chatgpt.site
- GitHub Pages：https://valentintti.github.io/agent-boundary-atlas/

## 本地运行

```bash
pnpm install
pnpm dev
```

生产构建：

```bash
pnpm build
```

产品资料与能力边界集中维护在 `app/agents.ts`。
